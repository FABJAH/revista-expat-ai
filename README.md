# Revista Expats AI

Landing + Backend con FastAPI y bots orquestadores para expatriados en Barcelona.

## Instalación
1. Clona el repositorio.
2. Instala dependencias con `pip install -r requirements.txt`.
3. Arranca el backend con `uvicorn backend.app:app --reload --port 8000`.
4. Abre el frontend con Live Server o `python3 -m http.server 5500`.

## Estructura
- frontend/ → HTML, CSS, JS
- backend/ → API con FastAPI
- bots/ → Bots especializados
- README.md → Documentación inicial
